# 提示词建议

## 基本结构

按需要选用，不必每项都填：

```text
主体：……
场景/背景：……
风格/媒介：摄影 / 插画 / 3D / ……
构图：全景 / 特写 / 俯视 / ……
光线与氛围：……
材质与细节：……
约束：不要文字、不要水印、……
```

## 丰富策略

- 用户提示词已经很具体：只做整理，不要擅自加角色、品牌、剧情。
- 用户提示词很短：可适度补摄影质感、清晰度、光线，仍要贴合用户意图。
- 编辑已有图：明确写「只改 X，保持 Y 不变」。

## 文生图示例

```text
一只橘色家猫趴在窗台下午睡，窗外是阴天城市，自然光，浅景深，写实摄影，无文字无水印
```

## 编辑示例

```text
只把背景换成暖色日落渐变，主体产品与边缘保持不变，无文字无水印
```
